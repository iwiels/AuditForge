# Skill: Browser API Mapping

### Protocol
1. **Runtime Capture**: Execute browser automation to capture XHR, Fetch, and WebSocket traffic in real-time.
2. **Session Analysis**: Identify how authentication tokens (cookies, headers) are passed and used.
3. **Data Flow Mapping**: Map how frontend interactions translate into backend API calls.
4. **Evidence**: Provide HAR files or refined JSON maps of the API surface.
