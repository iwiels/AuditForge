# Skill: Proxy Capture Replay

### Protocol
1. **Traffic Interception**: Use mitmproxy or similar tools to capture authenticated sessions.
2. **Replay Validation**: Verify that captured requests can be replayed or modified to yield different results.
3. **Sensitive Data Sanitization**: Mask PII or sensitive tokens before saving replay templates in the audit evidence.
4. **Automation**: Convert manual proxy captures into automated verification scripts.
