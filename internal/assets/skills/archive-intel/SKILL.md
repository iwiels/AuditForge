# Skill: Archive Intel

### Protocol
1. **Historical Mining**: Use Wayback Machine, AlienVault, and URLScan to find historical endpoints.
2. **Context Reconstruction**: Identify parameters that existed in previous versions and test if they still influence current logic (legacy IDORs).
3. **Evidence**: Save a list of "historical candidates" for verification.
4. **Delegation**: Send high-potential historical URLs to `security-scout` for current status validation.
