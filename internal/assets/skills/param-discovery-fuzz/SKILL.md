# Skill: Param Discovery Fuzz

### Protocol
1. **Parameter Discovery**: Identify unlinked or hidden HTTP parameters through manual traversal or proxy logs.
2. **Targeted Testing**: Apply targeted test cases for common vulnerabilities (SQLi, XSS, LFI) to discovered parameters.

3. **Rate Control**: Maintain strict rate limits to avoid triggering WAFs or causing DoS.
4. **Inheritance**: Assume authorization inherited from `security-web` specialized tasking.
